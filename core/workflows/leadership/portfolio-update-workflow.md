# Leadership And Portfolio Update Workflow

Purpose: Aggregate workspace-level state into leadership-ready portfolio and executive summaries.

## Inputs

- suite manifest
- workspace manifests
- status mail artifacts
- reliability and integration state

## Outputs

- `portfolio_update`
- `leadership_review`

## Rules

- preserve workspace traceability
- surface cross-workspace risks clearly
- avoid flattening all work into a single generic summary
