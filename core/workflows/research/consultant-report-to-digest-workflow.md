# Consultant Report To Digest Workflow

Purpose: Convert a dense consultant or analyst report into a reusable `consultant_report_digest` and note-card set.

## Inputs

- report file or normalized extraction
- PM research question

## Outputs

- `consultant_report_digest`
- `source_note_card`

## Rules

- extract only claims relevant to the PM question rather than summarizing the whole report
- keep thesis, reusable assets, strategic implications, contradictions, and caveats separately visible
- record where the report sample, timeframe, or methodology limits confidence
