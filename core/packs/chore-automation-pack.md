# Chore Automation Pack

Purpose: Automate recurring PM operational follow-up work so the PM can focus on judgment, strategy, and customer problems.

## Included capabilities

- follow-up queue management and prioritization
- weekly team update generation
- demo and sprint readout compilation
- follow-up tracking, escalation, and digest generation
- task creation from approved PRDs and story packs
- task status change detection and alerting
- release note first-draft generation
- recurring follow-up scheduling with cadence triggers
- automation learning from PM approval and edit patterns
